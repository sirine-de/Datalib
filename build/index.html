Welcome to the DataLib documentation!¶

DataLib is a Python library designed for data manipulation and analysis.

Features¶

  • Data manipulation: Load, clean, and transform data.

  • Statistics: Calculate basic statistics (mean, median, standard deviation) and perform statistical tests.

  • Visualization: Simple and advanced data visualizations.

  • Advanced analysis: Linear regression, clustering, and principal component analysis (PCA).

Modules¶

This section includes the documentation for all modules in DataLib.
